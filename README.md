# 🎓 UPAO App - Aula Virtual

Aplicación móvil desarrollada con **React Native + Expo Router** para gestión académica universitaria.

---

## 🚀 Características

- 🔐 Sistema de autenticación (login/logout)
- 📚 Aula virtual
- 📅 Horarios
- 📝 Calificaciones
- 🧾 Historial académico
- 👤 Cuenta de usuario
- 🌙 Modo oscuro / claro
- 💾 Persistencia de sesión

---

## 📱 Tecnologías

- Expo / React Native
- Expo Router
- AsyncStorage
- React Navigation
- TypeScript

---

## ⚙️ Instalación

```bash
npm install
npx expo start
